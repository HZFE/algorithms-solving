# solve-algorithms

记录群友的每日算法题

## 记录方式

仓库会每日新建一个 issue，如果使用 VSCode 刷题可以通过 [HZFE Algorithms](https://marketplace.visualstudio.com/items?itemName=gongpei.hzfe-algorithms) 插件，写完题之后一键同步到最新的 issue
